🧩 How to install your Chrome extension (unzipped)
1. Open Chrome extensions page
Open Chrome
Type this in the top bar:
chrome://extensions/
Press Enter
2. Turn ON Developer Mode
Look at the top right
Turn on “Developer mode”
3. Click “Load unpacked”
A button will appear
Click “Load unpacked”
4. Select your folder
Find your folder:
9AnimieTV-BingeWatch-Extension

👉 Click the folder itself (not the files inside)

Click Select Folder
5. Done ✅
The extension should now appear in the list

🔄 Final step (IMPORTANT)
Go to your website
Press F5 (refresh)
